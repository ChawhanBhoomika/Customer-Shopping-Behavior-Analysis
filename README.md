# Customer Shopping Behavior Analysis

This project analyzes real customer shopping behavior using **Python**, **SQL**, and **Power BI**.  
It includes exploratory data analysis (EDA), SQL-driven insights, and interactive dashboards to understand trends in spending, demographics, and purchasing behavior.

---

## 📌 Project Overview

This project explores a dataset of **3,900 customer transactions** with 18 attributes, including:

- Customer demographics  
- Transaction details  
- Spending patterns  
- Discount and promo usage  
- Payment methods  
- Shipping preferences  
- Customer loyalty metrics  

The goal is to uncover actionable insights that help businesses improve decision-making.

## 🔍 Exploratory Data Analysis (Python)

The EDA performed in Jupyter Notebook includes:

- Loading the dataset with pandas  
- Viewing top records (`.head()`)  
- Inspecting structure (`.info()`)  
- Summary statistics using `.describe()`  
- Checking missing values  
- Analyzing categorical distributions  
- Verifying data quality  

The dataset was clean and required minimal preprocessing.

---

## 🧮 SQL Analysis

The SQL script includes:

- Revenue by age group  
- Revenue by category  
- Top customers by total purchase amount  
- Payment method distribution  
- Discount usage patterns  
- Customer loyalty and frequency analysis  

These queries provided deep business insights.

## 📊 Power BI Dashboard

The Power BI report visualizes:

- Customer segmentation KPIs  

The dashboard offers a comprehensive view of customer behavior.

---

## ⭐ Key Insights

- Age group **25–45** generates the highest revenue  
- **Electronics** has the highest revenue; **Clothing** has most purchases  
- Majority of customers prefer **online/card payments**  
- High-frequency buyers are the most valuable customers  
- Discount users purchase frequently but spend less per order  

---

## 💡 Business Recommendations

- Implement loyalty programs for high-value customers  
- Use targeted promotions for occasional shoppers  
- Offer incentives for online payments  
- Improve fast shipping options for better satisfaction  
- Stock more products in trending categories
